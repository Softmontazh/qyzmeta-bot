## � О проекте

**Qyzmeta-Bot** — это инновационная платформа автоматизации управления многоквартирными домами через Telegram. Система объединяет жителей, управляющие компании и сервисные организации в единую экосистему для эффективного решения вопросов ЖКХ.

### 🎯 Основные возможности

- **Автоматизированный прием заявок** от жителей 24/7
- **Умная система распределения** заявок по исполнителям
- **Полная аналитика и отчетность** для управляющих компаний
- **Интегрированная система оценки качества** услуг
- **Многоуровневая система ролей** и прав доступа

## �️ База данных

Система использует PostgreSQL с 6 основными таблицами:

- **User** — пользователи системы (жители, сотрудники УК, админы)
- **JK** — жилищные комплексы  
- **UserJK** — связь пользователей с ЖК
- **Offer** — заявки от жителей
- **JKServiceProvider** — поставщики услуг для ЖК
- **Lot** — лоты для обмена файлами

Подробное описание: [DATABASE_MODELS_DOCUMENTATION.md](DOCS/DATABASE_MODELS_DOCUMENTATION.md)

## 🔧 Установка и настройка

### 1. Клонирование репозитория
```bash
git clone https://github.com/softmontazh/qyzmeta-bot.git
cd qyzmeta-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` на основе `.env.example`:
```env
# Telegram Bot Token (получить у @BotFather)
BOT_TOKEN=

# Database (настройте подключение к PostgreSQL)
DATABASE_URL=postgresql://username:password@localhost:5432/database_name

# Admin settings (Telegram ID администраторов через запятую)
ADMIN_IDS=

# File sharing (ID группы для обмена медиафайлами)
BUS_ID=
```

⚠️ **Важно**: Никогда не публикуйте реальные токены и пароли в открытом доступе!

### 4. Инициализация базы данных
```bash
# Создание таблиц
python -c "from database.engine import init_db; import asyncio; asyncio.run(init_db())"
```

### 5. Запуск бота
```bash
python app.py
