# 🏢 Система поставщиков услуг QYZMETA Bot

## Обзор системы

Система поставщиков услуг позволяет управляющим компаниям и администраторам ЖК добавлять и управлять различными поставщиками услуг для жильцов.

## Категории услуг (`OfferCategory`)

### Доступные категории

| Категория | Emoji | Русское название | Описание |
|-----------|-------|------------------|----------|
| `CLEANING` | 🧹 | Клининг | Услуги уборки |
| `REPAIR` | 🔧 | Ремонт | Ремонтные работы |
| `DELIVERY` | 🚚 | Доставка | Доставка товаров и продуктов |
| `BEAUTY` | 💄 | Красота | Косметологические услуги |
| `EDUCATION` | 🎓 | Образование | Обучение и курсы |
| `HEALTH` | 🏥 | Здоровье | Медицинские услуги |
| `FOOD` | 🍕 | Еда | Общественное питание |
| `TRANSPORT` | 🚗 | Транспорт | Транспортные услуги |
| `ENTERTAINMENT` | 🎭 | Развлечения | Развлекательные услуги |
| `OTHER` | 📦 | Другое | Прочие услуги |

## Структура данных

### Модель поставщика услуг (`JKServiceProvider`)

```python
class JKServiceProvider(Base):
    id: int                    # Уникальный ID
    jk_id: int                # ID ЖК (связь с JK)
    category: OfferCategory    # Категория услуг
    organization_name: str     # Название организации
    contact_person: str        # Контактное лицо
    phone: str                # Телефон
    email: str (optional)     # Email
    description: str          # Описание услуг
    is_active: bool           # Активность (по умолчанию True)
    created_at: datetime      # Время создания
```

## Функциональность

### Основные возможности

1. **Просмотр поставщиков** (`/service_providers_list`)
   - Список всех поставщиков по ЖК
   - Группировка по категориям
   - Показ только активных поставщиков

2. **Добавление поставщика** (`/add_service_provider`)
   - FSM-форма для ввода данных
   - Валидация обязательных полей
   - Автоматическая привязка к ЖК

3. **Управление поставщиками**
   - Активация/деактивация
   - Редактирование информации
   - Удаление записей

### FSM для добавления поставщика

```python
class ManageServiceProviderStates(StatesGroup):
    select_jk_for_add = State()          # Выбор ЖК
    select_category = State()            # Выбор категории
    enter_organization_name = State()    # Название организации
    enter_contact_person = State()       # Контактное лицо
    enter_phone = State()               # Телефон
    enter_email = State()               # Email (опционально)
    enter_description = State()         # Описание услуг
    confirm_provider = State()          # Подтверждение
```

## Права доступа

### Кто может управлять поставщиками

1. **Создатель бота** - полные права на все ЖК
2. **Администраторы платформы** - полные права на все ЖК
3. **Админы ЖК** - права только на свои ЖК (из `admins_list`)

### Проверка прав

```python
# Проверка глобальных прав
is_creator_by_env = is_creator_by_environment(user_id)
global_admin = user.role in [UserRole.ADMIN, UserRole.CREATOR]

# Проверка прав на конкретный ЖК
if is_creator_by_env or global_admin:
    available_jks = await orm_get_all_jks(session)  # Все ЖК
else:
    available_jks = await orm_get_jks_by_user_admin(session, user_id)  # Только свои ЖК
```
