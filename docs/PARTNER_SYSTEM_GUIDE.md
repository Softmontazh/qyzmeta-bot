# 🤝 Руководство по партнерской системе Qyzmeta-Bot

> **Полное руководство по системе управления партнерами: от подачи заявки до работы в панели**

## 📋 Обзор системы

Партнерская система Qyzmeta-Bot представляет собой комплексное решение для управления и координации деятельности партнеров платформы. Система включает автоматизированную подачу заявок, модерацию, контроль доступа и специализированную панель управления.

### ✨ Ключевые возможности

- 📝 **FSM-форма заявок** — пошаговый сбор данных
- ⚡ **Автоматическая модерация** — быстрое рассмотрение заявок
- 🛡️ **Контроль доступа** — проверка прав на уровне обработчиков
- 📊 **Партнерская панель** — специализированный интерфейс
- 🔄 **Управление статусами** — полный жизненный цикл заявки

---

## 🚀 Начало работы

### Для новых партнеров

1. **Откройте бота** [@qyzmetabot](https://t.me/qyzmetabot)
2. **Отправьте команду** `/is_partner`
3. **Нажмите кнопку** "📝 Стать партнером"
4. **Заполните форму** (5 шагов):
   - ФИО (полное имя)
   - Название компании/организации
   - Цель партнерства и планы
   - Контактный телефон
   - Подтверждение данных

### Процесс модерации

1. **Автоматическая подача** — заявка сохраняется с статусом `PENDING`
2. **Уведомление админов** — система отправляет уведомление создателю
3. **Рассмотрение** — администратор просматривает данные
4. **Принятие решения** — одобрение или отклонение
5. **Уведомление пользователя** — автоматическое сообщение о результате

## 👤 Кабинет партнера

Центральный раздел для управления партнерским профилем.

### Информация профиля

- **Статус заявки** — текущий статус с описанием
- **Личные данные** — ФИО, компания, телефон
- **Цель партнерства** — описание планов
- **Даты** — подача заявки и последнее обновление

### Доступные действия

| Действие | Кнопка | Описание |
|----------|--------|----------|
| **Редактировать** | ✏️ Редактировать заявку | Запрос изменения данных |
| **Удалить** | 🗑️ Удалить заявку | Запрос удаления из системы |
| **Назад** | ⬅️ Назад | Возврат к партнерской панели |

### Статусы заявок

| Статус | Отображение | Доступные действия |
|--------|-------------|-------------------|
| `PENDING` | ⏳ Ожидает рассмотрения | Редактирование, Удаление |
| `APPROVED` | ✅ Одобрена | Редактирование данных |
| `REJECTED` | ❌ Отклонена | Повторная подача |
| `EDIT_REQUEST` | ✏️ Запрос на редактирование | Ожидание модерации |
| `DELETE_REQUEST` | 🗑️ Запрос на удаление | Ожидание обработки |

## 🛡️ Безопасность и контроль доступа

### Уровни защиты

1. **Проверка заявки** — только одобренные партнеры
2. **Декораторы** — автоматическая проверка прав
3. **Валидация данных** — проверка на каждом этапе
4. **Аудит действий** — логирование всех операций

### Декоратор `@partner_access_required`

```python
def partner_access_required(func):
    """Проверяет наличие одобренной заявки партнера"""
    async def wrapper(message: Message, state: FSMContext, session: AsyncSession, **kwargs):
        if not await is_approved_partner(session, message.from_user.id):
            await message.answer("❌ У вас нет доступа к партнерской панели.")
            return
        return await func(message, state, session, **kwargs)
    return wrapper
```
