# 📋 Классификация команд Qyzmeta-Bot по назначению

> **Проект**: Qyzmeta-Bot - Telegram бот для цифровизации ЖКХ Казахстана  
> **Документ**: Подробная классификация команд по смыслу и назначению  
> **Дата**: 12 июля 2025 г.

---

## 🎯 **1. СИСТЕМА КОМАНД - ОБЩАЯ СТРУКТУРА**

### 🔄 **Жизненный цикл пользователя:**
```
ГОСТЬ → РЕГИСТРАЦИЯ → ПОЛЬЗОВАТЕЛЬ → РЕЗИДЕНТ ЖК → АКТИВНЫЙ УЧАСТНИК
```

### 🏗️ **Архитектура доступа:**
- **Уровень 0:** Публичные команды (доступны всем)
- **Уровень 1:** Команды для зарегистрированных пользователей  
- **Уровень 2:** Команды для резидентов ЖК
- **Уровень 3:** Административные команды

## 🚀 **2. КОМАНДЫ ПЕРВОГО КОНТАКТА**

### **Назначение:** Знакомство с ботом и начальная настройка

| Команда | Классификация | Детальное назначение |
|---------|---------------|---------------------|
| `/start` | **Точка входа** | • Первая команда для новых пользователей<br>• Регистрация в системе как GUEST<br>• Запрос контакта для активации<br>• Переход в основной интерфейс для зарегистрированных |
| `/help` | **Справочная** | • Руководство по использованию бота<br>• Описание основных функций<br>• Инструкции для новичков<br>• Решение типовых проблем |

### **Бизнес-логика:**
- `/start` - главная воронка конверсии GUEST → USER
- `/help` - снижение нагрузки на поддержку через самообслуживание

## 🛒 **6. КОМАНДЫ ТОРГОВОЙ ПЛАТФОРМЫ**

### **Назначение:** P2P торговля между жителями и соседский обмен

| Команда | Классификация | Детальное назначение |
|---------|---------------|---------------------|
| `/market` | **Обзорная** | • Просмотр всех активных объявлений<br>• Фильтрация по категориям<br>• Контакты продавцов<br>• Быстрый поиск товаров |
| `/search_lots` | **Поисковая** | • Расширенный поиск объявлений<br>• Фильтры по цене, типу, городу<br>• Сохранение поисковых запросов<br>• Уведомления о новых подходящих лотах |
| `/exit_search` | **Навигационная** | • Выход из режима поиска<br>• Возврат в основное меню<br>• Сброс поисковых фильтров<br>• Завершение FSM состояния |

### **Типы объявлений:**
- **BUY** - Купить (ищу товар)
- **SELL** - Продать (предлагаю товар)  
- **EXCHANGE** - Обмен (меняю на что-то)
- **GIVEAWAY** - Отдам даром (бесплатно)
- **RENT** - Арендую (ищу в аренду)
- **RENTOUT** - Сдам в аренду (предлагаю аренду)
- **REQUEST** - Запрос (нужна услуга)
- **SUGGEST** - Предложение (предлагаю услугу)

### **Бизнес-логика:**
- Развитие соседских отношений
- Экономия ресурсов через повторное использование
- Монетизация через комиссии (в будущем)

## 📊 **10. СИСТЕМЫ ОГРАНИЧЕНИЙ И ДОСТУПА**

### **🔒 Уровни доступа к командам:**

#### **Публичные (Уровень 0):**
- `/start`, `/help`, `/policy`, `/support`
- **Цель:** Максимальная доступность

#### **Зарегистрированные (Уровень 1):**
- `/my_profile`, `/settings`, `/main_menu`, `/delete_profile`
- **Требование:** Роль ≠ GUEST + наличие телефона

#### **Резиденты ЖК (Уровень 2):**
- `/my_jk`, `/add_my_jk`, `/add_offer`, `/market`, `/search_lots`
- **Требование:** Привязка к ЖК

#### **Администраторы (Уровень 3):**
- `/admin`, `/exit_admin`, `/manage_jk`, `/add_jk`
- **Требование:** ID в PLATFORM_ADMIN_IDS

### **🎯 Лимиты по ролям:**

| Роль | Лимит лотов | Доступные функции |
|------|-------------|-------------------|
| **GUEST** | 0 | Только просмотр + регистрация |
| **USER** | 10 | Базовый функционал |
| **ADMIN** | 100 | + модерация |
| **MANAGER** | 1,000 | + управление ЖК |
| **PARTNER** | 10,000 | + коммерческие функции |
| **OWNER** | ∞ | + владельческие права |
| **CREATOR** | ∞ | Полный доступ |

## 🔄 **11. FSM СОСТОЯНИЯ И ПЕРЕХОДЫ**

### **Многошаговые процессы:**

#### **Создание заявки (AddOffer):**
1. `choose_jk` → Выбор ЖК (если несколько)
2. `choose_category` → Выбор категории проблемы  
3. `set_title` → Ввод названия заявки
4. `set_description` → Описание проблемы
5. `set_media` → Прикрепление фото/видео (опционально)
6. `confirm` → Подтверждение и отправка

#### **Добавление ЖК (JKCreationState):**
1. `set_name` → Название ЖК
2. `set_address` → Адрес (город, улица, дом)
3. `set_details` → Дополнительные данные
4. `confirm` → Подтверждение создания

#### **Поиск лотов (SearchLot):**
1. `set_criteria` → Критерии поиска
2. `filter_results` → Применение фильтров
3. `view_results` → Просмотр результатов

## 📈 **12. МЕТРИКИ И АНАЛИТИКА**

### **Ключевые показатели по командам:**

#### **Конверсионные метрики:**
- **Регистрация:** `/start` → отправка контакта (%)
- **Активация:** регистрация → первая заявка (%)
- **Вовлечение:** активные пользователи / месяц

#### **Продуктовые метрики:**
- **Популярность команд:** частота использования
- **Категории заявок:** распределение по типам
- **Время решения:** от создания до закрытия заявки

#### **Бизнес-метрики:**
- **Монетизация:** доход с торговой площадки
- **Retention:** возврат пользователей
- **Virality:** приглашения новых пользователей

## 🎯 **13. СТРАТЕГИЧЕСКИЕ ЦЕЛИ КОМАНД**

- Мобильное приложение-компаньон

### **Долгосрочные (1-3 года):**
- Экосистема умного города
- Интеграция с государственными сервисами
- Франчайзинговая модель для других регионов
