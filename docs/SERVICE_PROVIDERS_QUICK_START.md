# 🚀 Быстрый старт: Поставщики услуг

## Для администраторов ЖК

### 1. Получение прав доступа

**Вариант А: Через заявку на роль Партнер**
1. В боте нажмите "🎯 Подать заявку на роль"
2. Выберите "🤝 Партнер"
3. Заполните форму:
   - ФИО
   - Название компании
   - Цель: "Управление ЖК [название]"
   - Контактный телефон
4. Дождитесь одобрения от администрации

**Вариант Б: Через назначение администратором**
- Обратитесь к создателю бота для назначения админом конкретного ЖК

### 2. Доступ к системе поставщиков

После получения прав:
1. В главном меню выберите секцию управления
2. Найдите кнопку "Поставщики услуг"
3. Откроется меню управления поставщиками

## Добавление поставщика услуг

### Шаг 1: Начало процесса
```
Меню → Поставщики услуг → ➕ Добавить поставщика услуг
```

### Шаг 2: Выбор ЖК (если у вас несколько)
- Выберите ЖК из списка доступных
- Если ЖК только один - пропускается автоматически

### Шаг 3: Выбор категории услуг
- 🧹 Клининг
- 🔧 Ремонт  
- 🚚 Доставка
- 💄 Красота
- 🎓 Образование
- 🏥 Здоровье
- 🍕 Еда
- 🚗 Транспорт
- 🎭 Развлечения
- 📦 Другое

### Шаг 4: Заполнение информации

**Обязательные поля:**
1. **Название организации** (мин. 2 символа)
   - Пример: `ООО "Чистый дом"`

2. **Контактное лицо** (мин. 2 символа)
   - Пример: `Иванов Иван Иванович`

3. **Телефон** 
   - Пример: `+7 777 123 45 67`

4. **Описание услуг** (мин. 10 символов)
   - Пример: `Профессиональная уборка квартир и офисов. Используем экологически чистые средства. Работаем 24/7.`

**Опциональные поля:**
5. **Email** (можно пропустить)
   - Пример: `info@cleanhouse.kz`

### Шаг 5: Подтверждение
- Проверьте все данные
- Нажмите "✅ Сохранить" для добавления

## Управление существующими поставщиками

### Редактирование информации
1. Найдите поставщика в списке
2. Выберите "✏️ Редактировать"
3. Измените нужные поля
4. Сохраните изменения

### Деактивация поставщика
1. Выберите поставщика
2. Нажмите "❌ Деактивировать"
3. Подтвердите действие

*Деактивированные поставщики не отображаются в общем списке, но остаются в базе данных*

### Повторная активация
1. В настройках выберите "Показать неактивных"
2. Найдите нужного поставщика
3. Нажмите "✅ Активировать"

## Типичные сценарии использования

### Сценарий 1: Новый ЖК
1. Получить права админа ЖК
2. Добавить основные категории поставщиков:
   - Клининг (уборка подъездов)
   - Ремонт (аварийные службы)
   - Доставка (продукты/лекарства)
3. Проинформировать жильцов о доступных услугах

### Сценарий 2: Сезонные услуги
- Летом: добавить услуги по уходу за придомовой территорией
- Зимой: добавить снегоуборочные службы
- Перед праздниками: добавить организаторов мероприятий

### Сценарий 3: Расширение услуг
1. Опросить жильцов о потребностях
2. Найти подходящих поставщиков
3. Добавить в систему с подробным описанием
4. Уведомить жильцов о новых возможностях

## Советы и лучшие практики

### ✅ Рекомендации:

1. **Подробные описания**
   - Указывайте время работы
   - Перечисляйте конкретные услуги
   - Добавляйте особенности (24/7, эко-средства, и т.д.)

2. **Актуальные контакты**
   - Проверяйте телефоны перед добавлением
   - Обновляйте информацию при изменениях
   - Добавляйте несколько способов связи

3. **Правильные категории**
   - Выбирайте наиболее подходящую категорию
   - Если не подходит ни одна - используйте "📦 Другое"

### ❌ Избегайте:

1. **Дублирования**
   - Не добавляйте одного поставщика дважды
   - Проверяйте список перед добавлением

2. **Неактуальной информации**
   - Не оставляйте старые номера телефонов
   - Удаляйте/деактивируйте закрывшиеся компании

3. **Слишком общих описаний**
   - Избегайте фраз типа "оказываем услуги"
   - Будьте конкретными в описании
