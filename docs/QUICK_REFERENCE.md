# ⚡ LotBox Platform - Быстрая справка

## 🎯 Что это?
**LotBox** - революционная платформа цифровизации стройки "снизу вверх"
**Qyzmeta** - облегченный ЖКХ интерфейс для жителей  
**MicrofirmKZ** - CRM для малого бизнеса + автогенерация документов

## 🔥 Ключевая идея
**Лот** = универсальная единица (задача/товар/заявка/что угодно)  
**UUID связи** = цифровая история объекта (основа WEB3.0)  
**Автопривязки** = заявки автоматически попадают нужным исполнителям

## 📂 Текущий статус
- ✅ **qyzmeta-bot** - базовый MVP готов (этот проект)
- 🔄 **lotbox-bot** - в отдельном репозитории  
- 🔄 **microfirm-kz** - в отдельном репозитории

## 🛠️ Что делаем дальше?
1. **Завершаем MVP Qyzmeta** (привязки каналов/УК/уведомления)
2. **Создаем shared модуль** с общими моделями  
3. **Интегрируем все проекты** через единую БД

## 📋 Для разработчика
- Читай `LOTBOX_PLATFORM_MANIFEST.md` - полная концепция
- Читай `DEVELOPMENT_STRATEGY.md` - план реализации
- Используй UUID везде для будущих связей
- Помни: заявки ЖКХ = тоже лоты в общей системе
