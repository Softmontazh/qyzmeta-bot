#### 1. Удален файл `system_info.py`

- Информация была рассчитана на веб-среду, а не на чат-сообщения

- Исправлен обработчик `support_cmd()` - теперь показывает `support_text` вместо `about_bot`

#### 3. Удалены веб-ссылки из статических файлов
**Из `static/help.py`:**
