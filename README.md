## 🎯 О проекте

**Qyzmeta-Bot** — инновационная платформа автоматизации управления многоквартирными домами через Telegram. Система объединяет жителей, управляющие компании и сервисные организации в единую экосистему для эффективного решения вопросов ЖКХ.

### ✨ Основные возможности

- **🚀 Автоматизированный прием заявок** от жителей 24/7
- **🎯 Умная система распределения** заявок по исполнителям
- **📊 Полная аналитика и отчетность** для управляющих компаний
- **⭐ Интегрированная система оценки качества** услуг
- **👥 Многоуровневая система ролей** и прав доступа
- **🤝 Система управления партнерами** с автоматической модерацией
- **📝 FSM-форма подачи заявок** на партнерство
- **📸 BUS Media System** — уникальная система обмена медиафайлами
- **💰 Система подписок с динамическим ценообразованием**
- **📈 Интегрированная аналитика монетизации** с отслеживанием конверсии

## 🛠 Установка для разработчиков

### 1. Клонирование репозитория
```bash
git clone https://github.com/softmontazh/qyzmeta-bot.git
cd qyzmeta-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` на основе `.env.example`:
```env
# Telegram Bot Token (получить у @BotFather)
BOT_TOKEN=your_bot_token_here

# Database (настройте подключение к PostgreSQL)
DATABASE_URL=postgresql://username:password@localhost:5432/database_name

# Admin settings (Telegram ID администраторов)
ADMIN_IDS=123456789,987654321

# File sharing (ID группы для обмена медиафайлами)
BUS_ID=your_bus_group_id
```

⚠️ **Важно**: Никогда не публикуйте реальные токены и пароли!

### 4. Инициализация базы данных
```bash
python -c "from database.engine import init_db; import asyncio; asyncio.run(init_db())"
```

### 5. Запуск бота
```bash
python app.py
```

## 🗄️ База данных

Система использует PostgreSQL с 7 основными таблицами:

- **User** — пользователи системы (жители, сотрудники УК, админы)
- **JK** — жилищные комплексы  
- **UserJK** — связь пользователей с ЖК
- **Offer** — заявки от жителей
- **JKServiceProvider** — поставщики услуг для ЖК
- **PartnerApplication** — заявки на партнерство с статусами
- **UserSubscription** — подписки пользователей с тарифами
- **SubscriptionPrice** — динамическое ценообразование с историей изменений
- **Lot** — лоты для обмена файлами

Подробное описание: [DATABASE_MODELS.md](docs/DATABASE_MODELS.md)

## � Система подписок и монетизации

### 🎯 Тарифные планы

| Тариф | Лимит адресов | Стоимость | Особенности |
|-------|---------------|-----------|-------------|
| 🆓 **Бесплатный** | 1 адрес | 0 ₸/мес | Базовые уведомления |
| ⭐ **Базовый** | 3 адреса | от 2,990 ₸/мес | Приоритетные уведомления, поддержка в чате |
| 💎 **Премиум** | 10 адресов | от 4,990 ₸/мес | Эксклюзивные предложения, персональный менеджер, аналитика |
| 👑 **VIP** | Безлимит | от 9,990 ₸/мес | VIP статус, приоритет во всех сделках, ранний доступ |

### 📊 Система управления ценами

- **Динамическое ценообразование** — изменение цен в реальном времени через админ-панель
- **История изменений** — полный аудит всех изменений цен с отметками времени
- **Валидация цен** — автоматическая проверка корректности цен (0-100,000 ₸)
- **Аналитика доходов** — подробная статистика по подпискам и конверсии

### 🔧 Администрирование

Управление ценами доступно через команду `/admin` → "Управление подписками":

- **💰 Управление ценами** — изменение стоимости тарифов
- **📜 История изменений** — просмотр всех изменений цен
- **📊 Аналитика подписок** — статистика пользователей и доходов
