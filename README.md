
# 🏢 Qyzmeta-Bot — цифровая инфраструктура ЖКХ

**Qyzmeta-Bot** — это Telegram-бот, позволяющий жильцам взаимодействовать с управляющими компаниями, обслуживающими организациями и цифровыми сервисами, связанными с домом.  
Он обеспечивает удобную форму подачи заявок, регистрации жильцов, хранения истории обращений и цифрового сопровождения объектов ЖКХ.
Проект создается как часть цифровой инфраструктуры в рамках платформы **LotBox** и станет частью масштабируемой экосистемы для ЖКХ.

---

## ⚙️ Технологический стек

- 🧠 **Aiogram 3.x** — асинхронный Telegram-фреймворк  
- 🐘 **SQLAlchemy 2.x** — ORM для PostgreSQL  
- 🐍 **Python 3.11+**
- 🔗 **PostgreSQL** — СУБД
- 🗂️ **FSM-состояния** — для обработки пошаговых сценариев  
- ☁️ **VPS-сервер** с Webhook для работы 24/7 (после завершения MVP)

## 🚀 Установка и запуск (dev-режим)

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` со следующими переменными окружения:

```
BOT_TOKEN=ваш_токен_бота
DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/qyzmeta
```

3. Запустите бота:

```bash
python app.py
```

> 💡 Убедитесь, что PostgreSQL запущен и база данных создана.
